# news-recap

Сейчас `news-recap` — это MVP с CLI для:

- сбора новостей из RSS/Atom,
- нормализации и очистки текста статей,
- семантической дедупликации и кластеризации,
- сохранения истории запусков и артефактов в SQLite.

Это технический ingestion-слой продукта. Пользовательский интерфейс без CLI будет в следующих
итерациях.

## Текущий Scope (Epic 1)

- Источник: RSS/Atom фиды (включая Inoreader Output RSS).
- Хранилище: локальный SQLite (по умолчанию `.news_recap.db`).
- Результат: запуски ingestion, нормализованные статьи, dedup-кластеры, примеры дублей.
